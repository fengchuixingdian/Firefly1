>[!TIP]
>
>Firefly is a fresh, beautiful, and modern personal blog theme template based on the Astro framework and the Fuwari template, designed for tech enthusiasts and content creators. It integrates a modern web tech stack, offering rich feature modules and a highly customizable interface so you can easily build a professional and visually appealing personal blog.
>
>In terms of key layouts, Firefly innovatively adds dual sidebars, an article grid layout, and a masonry layout.
>
>It also adds practical widgets such as site statistics, calendar, table of contents, music player, and quick category navigation, enriching page layout.
>
>Meanwhile, it supports share posters, related post recommendations and random posts to enrich article content.
>
>**If you reference or use components and codes from Firefly, please give appropriate credit.**
>
>Firefly also retains the original fuwari layout, which can be switched freely via configuration files.
>
>Firefly supports i18n multilingual UI. Languages other than Simplified Chinese are translated by AI. Feel free to submit Pull Requests to correct mistakes.

## ✨ Features

### Core Features
- [x] **Astro + Tailwind CSS** - High-performance static site generation
- [x] **Smooth Animations** - Elegant page transition effects
- [x] **Responsive Design** - Fully adapted for all devices
- [x] **Multi-language Support** - Built-in internationalization system
- [x] **Full-text Search** - Fast client-side content search

### Personalization
- [x] Flexible single / dual sidebar layout
- [x] Multiple article display layouts
- [x] Custom global font settings
- [x] Fully editable footer content
- [x] Light / Dark / System follow display modes
- [x] Fully customizable navigation bar
- [x] Various wallpaper background styles
- [x] Freely adjustable theme color tones

You are welcome to submit Pull Requests to add new features and optimize experiences.
